# P plus plus plus

This repository collects seminar material about coding-related projects. The current focus is stepwise refinement: starting from a clear specification, making explicit design decisions, and refining the solution into executable code.

## Current seminar

- [2026-04-20 Stepwise Refinement](seminars/2026-04-20-stepwise-refinement/README.md)

## Running the example

```bash
uv run python seminars/2026-04-20-stepwise-refinement/compact_solver.py
uv run python seminars/2026-04-20-stepwise-refinement/board_solver.py
```

The current example uses the eight queens problem to show how a natural-language specification can serve as one refinement stage on the way to executable Python.
